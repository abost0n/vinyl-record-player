# vinyl-record-player

Repository
(https://github.com/abost0n/vinyl-record-player.git)

Description
An interactive digital vinyl record player that allows users to upload and play their own music while displaying a spinning record animation. This project combines audio playback with visual design to create a digital media experience.

Features
Vinyl Record Animation

A record player spins using animation while music plays.
Music Upload System

Users upload their own audio files.
Play and Pause Controls

Buttons control the music.
Needle Arm Animation

A needle moves when the music starts.
Album Cover Display

Users upload an image for the record center.
Volume Control

Slider controls volume.
Scratch Effect

Users drag the record to simulate scratching.
Challenges

Syncing animation with music.
Designing the vinyl look.
